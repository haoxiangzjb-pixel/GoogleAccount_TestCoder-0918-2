# Exemplos de Blocos de Código em Markdown

## Código em Python

```python
def hello_world():
    print("Olá, mundo!")
    for i in range(5):
        print(f"Número: {i}")

hello_world()
```

## Código em JavaScript

```javascript
function sayHello() {
    console.log("Hello, world!");
    for (let i = 0; i < 5; i++) {
        console.log(`Número: ${i}`);
    }
}

sayHello();
```

## Código em HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Exemplo HTML</title>
</head>
<body>
    <h1>Olá, mundo!</h1>
    <p>Este é um exemplo de código HTML.</p>
</body>
</html>
```
