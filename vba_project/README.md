# Excel Sheet Formatter VBA Macro

This project contains a VBA macro that formats an Excel sheet with the following features:

- Formats headers with bold white text on a dark background
- Adds borders to the data range
- Auto-fits columns for better readability
- Adds sample data for demonstration

## How to use

1. Open Excel and press Alt+F11 to open the VBA editor
2. Insert a new module
3. Copy and paste the code from `format_sheet.vba`
4. Run the macro by pressing F5 or from Excel's macro menu

The macro will format the active sheet with sample data.
