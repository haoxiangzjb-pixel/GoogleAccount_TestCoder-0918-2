# Process List Scripts

This directory contains scripts to get a list of processes and save them to a file with a random name.

## Files

- `get_processes.ps1`: PowerShell script to get processes and save to a random-named CSV file
- `get_processes_txt.ps1`: PowerShell script to get processes and save to a random-named text file
- `get_processes_python.py`: Python script that performs the same function as the PowerShell scripts
- `processes_*.csv`: Example output file generated by the Python script

## Usage

### PowerShell
Run either of the .ps1 files in a PowerShell environment:
```powershell
.\get_processes.ps1
```

### Python
Run the Python script in a Python environment with psutil installed:
```bash
python get_processes_python.py
```
