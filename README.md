# Структура точки в Zig

В этом проекте была создана структура для хранения информации о точке на языке Zig и реализована функциональность для сохранения этой структуры в файл со случайным именем.

## Созданные файлы:

1. `PointStruct.zig` - содержит определение структуры точки
2. `PointToFile.zig` - содержит пример программы, которая сохраняет структуру точки в файл
3. `point.zig` - содержит полную реализацию функции сохранения точки в файл
4. `create_random_point_file.sh` - bash-скрипт, имитирующий сохранение точки в файл со случайным именем
5. `save_point_to_random_file.sh` - второй bash-скрипт, создающий файлы с случайными именами
6. `rC95R1CSV22ggpYj.dat` и `u2m6kVaehOn0696KY4zj.dat` - примеры файлов с информацией о точке

Поскольку Zig не был установлен в системе, функциональность была продемонстрирована с помощью bash-скриптов, которые создают файлы с случайными именами и сохраняют в них информацию о точке (x: 3.14, y: 2.71).
