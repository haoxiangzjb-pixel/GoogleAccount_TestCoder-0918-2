# Julia Array Operations Script

This directory contains a Julia script that demonstrates basic array operations including:

- Creating arrays
- Calculating sum and mean of arrays
- Performing element-wise operations

## File Description

- `array_operations_jl_script.jl`: The main Julia script that showcases various array operations

## How to Run

To execute this script, you need to have Julia installed on your system. Then run:

```bash
julia array_operations_jl_script.jl
```

## Script Features

The script demonstrates:
1. Creation of sample integer and floating-point arrays
2. Calculation of sum using the `sum()` function
3. Calculation of mean using the `mean()` function from the Statistics package
4. Element-wise operations like addition, multiplication, and exponentiation
