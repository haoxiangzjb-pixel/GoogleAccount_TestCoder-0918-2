# PowerShell Date Time Script

This directory contains a PowerShell script that outputs the current date and time.

## Files

- `datetime_output.ps1` - The PowerShell script that displays the current date and time

## Requirements

- PowerShell 5.1 or later (Windows)
- PowerShell 7.x or later (Cross-platform)

## How to Run

1. Open a PowerShell terminal
2. Navigate to this directory
3. Execute the script:

```powershell
.\datetime_output.ps1
```

## Expected Output

The script will display:
- Current date and time
- Formatted date (MM/dd/yyyy)
- Formatted time (HH:mm:ss)

## Note

This script was created as part of a coding exercise and may not run in all environments.
